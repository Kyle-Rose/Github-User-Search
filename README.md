## 🧠 Project Overview  
This is a simple web application that allows users to search for GitHub profiles and view key information such as avatar, followers, and public repositories.  

The app demonstrates working with external APIs, asynchronous JavaScript, and dynamic DOM updates.

---

## 🚀 Features  
- Search for GitHub users by username  
- Display profile avatar  
- Show follower count  
- Show number of public repositories  
- Error handling for invalid usernames  

---

## 🛠️ Tech Stack  
- HTML  
- CSS  
- JavaScript  
- GitHub REST API  

---

## ⚙️ How It Works  
The app sends a request to the GitHub API: https://api.github.com/users/{username}

It then dynamically updates the page with the returned user data using JavaScript.

---

## 📂 Project Structure  
- `index.html` – Page structure  
- `style.css` – Styling  
- `script.js` – API calls and DOM manipulation
